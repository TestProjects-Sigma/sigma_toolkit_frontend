                   Sigma's Toolkit Launcher - Portable Version 
================================================================================ 
 
Version: Portable Python 3.11.9 
Created: 11/07/2025 22:01:58.52 
 
FILES: 
------- 
Launch_Sigma_Toolkit.bat    - Main launcher for the toolkit 
Setup_Repair.bat           - Fix/reinstall dependencies 
Install_App_Requirements.bat - Install requirements for all apps 
Python_Shell.bat           - Open Python command prompt 
main.py                    - Main toolkit application 
README.md                  - Documentation 
 
FOLDERS: 
-------- 
python\                    - Portable Python installation 
apps\                      - Your Python applications 
 
USAGE: 
------ 
1. Double-click 'Launch_Sigma_Toolkit.bat' to start the toolkit 
2. Add your Python apps to the 'apps' folder 
3. Use 'Setup_Repair.bat' if you encounter issues 
4. Use 'Install_App_Requirements.bat' to batch-install app dependencies 
 
REQUIREMENTS: 
------------- 
- Windows 7 or later 
- No system Python installation required 
- Approximately 150MB of disk space 
 
This portable version can be moved to any location or USB drive. 
All settings and apps will be preserved.
